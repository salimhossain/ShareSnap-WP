== Description ==

ShareSnap WP is a powerful WordPress plugin that transforms your blog posts into eye-catching social media posters. Perfect for sharing on Facebook, Instagram, Twitter, and other social platforms.

= Key Features =

* **Automatic Poster Generation** - Convert any WordPress post into a shareable poster
* **Customizable Design** - Full control over colors, fonts, and layout
* **Background Images** - Upload custom backgrounds or use defaults
* **Featured Image Support** - Automatically includes post featured images
* **Flexible Typography** - Adjust font size, line height, and color
* **Position Controls** - Place heading at top or bottom
* **Image Positioning** - 9 different position options for featured images
* **Zoom Controls** - Scale featured images from 10% to 500%
* **Branding** - Add your website URL and custom details
* **Easy Download** - One-click download of high-quality posters
* **Meta Box Integration** - Quick access from post editor sidebar
* **Multiple Access Points** - Available in main menu, settings, and tools

= Perfect For =

* News websites and blogs

= How It Works =

1. Create or edit a WordPress post
2. Click "Get ShareSnap Poster" from the post editor sidebar
3. Customize the poster with your preferred settings
4. Download and share on social media

== Installation ==

= Automatic Installation =

1. Log in to your WordPress admin panel
2. Navigate to Plugins > Add New
3. Search for "ShareSnap WP"
4. Click "Install Now" and then "Activate"

= Manual Installation =

1. Download the plugin zip file
2. Log in to your WordPress admin panel
3. Navigate to Plugins > Add New > Upload Plugin
4. Choose the downloaded zip file and click "Install Now"
5. Activate the plugin

= After Installation =

1. Go to ShareSnap WP in your admin menu
2. Configure default settings (background, colors, etc.)
3. Create or edit a post
4. Click "Get ShareSnap Poster" from the sidebar meta box
5. Customize and download your poster

== Frequently Asked Questions ==

= How do I create a poster for my post? =

From the post editor, look for the "ShareSnap" meta box in the sidebar. Click "Get ShareSnap Poster" to open the poster generator.

= Can I customize the poster design? =

Yes! You have full control over background images, text colors, font sizes, image positioning, and more.

= What image formats are supported? =

The plugin generates posters as PNG images with transparent backgrounds where applicable.

= Can I use custom fonts? =

The plugin uses your WordPress theme's fonts. You can add custom fonts through your theme.

= Will this work with any WordPress theme? =

Yes! ShareSnap WP is theme-independent and works with any properly coded WordPress theme.

= Can I save my settings? =

Yes! Click "Save Settings" to save your preferences. They'll be used as defaults for future posters.

= How do I reset to default settings? =

Click the "Reset" button in the controls panel to restore all default settings.

= Is there a limit to poster size? =

Posters are generated at optimal sizes for social media sharing (typically 1080x1080 or 1200x630).

= Can I edit the poster after downloading? =

Yes! Download the poster and edit it with any image editing software like Photoshop, GIMP, or Canva.

= Does it support multiple languages? =

Yes! The plugin is translation-ready and supports WordPress internationalization.

= Will this slow down my website? =

No! The plugin only loads resources on its admin pages and has minimal impact on performance.

= Can I use this for commercial purposes? =

Yes! The plugin is licensed under GPL v2, which allows commercial use.
